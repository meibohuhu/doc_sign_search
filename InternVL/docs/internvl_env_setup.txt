InternVL Conda Environment Setup
================================

The commands below recreate the `internvl` conda environment used for fine-tuning
`OpenGVLab/InternVL2_5-4B` on the How2Sign dataset. Run them from the project
root (`/home/mh2803/projects/sign_language_llm`) on a login node.


0. Load toolchains (per session)
--------------------------------
```
spack load /lhqcen5                 # GCC toolchain / common deps
spack load cuda@12.4.0/obxqih4      # CUDA toolkit (provides nvcc headers/libs)
export CUDA_HOME=$(spack location -i cuda@12.4.0/obxqih4)
export LD_LIBRARY_PATH="$CUDA_HOME/lib64:${LD_LIBRARY_PATH:-}"
```


1. Create & activate the conda environment
------------------------------------------
```
conda create -y -n internvl python=3.10.13
source /home/mh2803/miniconda3/bin/activate internvl
python -m pip install --upgrade pip setuptools wheel
```


2. Install PyTorch with CUDA 12.1 wheels
---------------------------------------
```
pip install torch==2.3.0+cu121 torchvision==0.18.0+cu121 torchaudio==2.3.0+cu121 \
    --index-url https://download.pytorch.org/whl/cu121
```


3. Core InternVL dependencies
-----------------------------
```
pip install -r InternVL/requirements/internvl_chat.txt
```

This pulls in (among others):
- accelerate < 1.0
- bitsandbytes==0.42.0
- decord
- deepspeed>=0.13.5
- einops==0.6.1 / einops-exts==0.0.4
- huggingface_hub, transformers==4.37.2, tokenizers==0.15.1
- numpy==1.26.4, scipy, scikit-learn>=1.2.2
- opencv-python, imageio, timm==0.9.12
- peft==0.10.0, shortuuid, tensorboardX, yacs


4. Extra packages required for our run
--------------------------------------
```
# Dataset utilities
pip install datasets==2.20.0 orjson==3.10.6

# Packed-sequence training extras
pip install flash-attn==2.5.7 --no-build-isolation --no-cache-dir

# MMCV prebuilt wheel (avoids from-source build failures)
mim install "mmcv==2.1.0" -f https://download.openmmlab.com/mmcv/dist/cu121/torch2.3/index.html

# Optional but recommended
pip install accelerate==0.30.1 packaging==24.1
```

Notes:
- `flash-attn` needs the CUDA headers; ensure `nvcc -V` resolves via `CUDA_HOME`.
- `mim` is bundled with the `mmengine` suite; if `mim` is missing run `pip install openmim`.


5. Verify installs
------------------
```
python - <<'PY'
import torch, flash_attn
import transformers, decord, datasets
print("Torch:", torch.__version__)
print("FlashAttention:", flash_attn.__version__)
print("Transformers:", transformers.__version__)
print("Decord:", decord.__version__)
print("Datasets:", datasets.__version__)
PY
```

The environment is now ready for the SLURM scripts under
`InternVL/scripts/finetune_internvl2_5_how2sign_*.sh`.
